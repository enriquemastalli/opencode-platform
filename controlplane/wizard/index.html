OpenCode Platform Setup

1. Company Domain

Configure the corporate domain where OpenCode will be hosted.

OpenCode will be available at opencode.[your-domain].

2. Cloudflare API

We need access to Cloudflare to configure DNS and Tunnels.

Requires Zone.DNS, Account.Cloudflare Tunnel, and Account.Access permissions.

3. Zero Trust Access

Restrict access to users matching your corporate email domains.

4. GitHub Integration

Connect the base repository where work will be performed.

5. AI Model Providers

Configure the API keys for the AI models.

6. Ready to Configure

Click Finish to apply the configuration. This will:

  • Create Cloudflare DNS & Tunnel
  • Setup Zero Trust Policies
  • Clone the GitHub repository
  • Start OpenCode Web securely
